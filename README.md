# Dotfiles
This holds the configs for the things I use. 

Mostly, neovim setup, alacritty setup, .zsh file, and more.

Note that this uses GNU Stow, and should be installed in the home directory. Then you can symlink the files stored here to the home folder really easily. 

ex: `stow nvim` - this will sym link the nvim package here to the home directory so neovim can see the config and use it. Very cool.

### Things that need to be installed:
- Homebrew
- Oh-My-Zsh
- SDK Man for managing java
- A nerd font, specifically `JetBrainsMono Nerd Font Mono` (The Regular Version)
- Node needs to be installed for some packages and neovim plugins
